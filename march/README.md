March Meeting: The Notional Machine & Function-First Programming Instruction
============================================================================

During the March meeting, Prof. Ashley Ringer-McDonald shared her expertise
on function-first programming strategies that explicitly introduce and support
student development of a _notional machine_---the mental code interpreter that
programmers use to pre-evaluate code to predict and understand its behavior.
